# websiteJs

A simple website built with HTML, JavaScript, and CSS — an early web development practice project.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Features

- Basic webpage layout and structure
- Interactive elements powered by JavaScript
- Custom styling with CSS

## Getting Started

Clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/chandurao77/websiteJs.git
cd websiteJs
open index.html
```
